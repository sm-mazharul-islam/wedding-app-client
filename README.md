# 💍 Elegance Wedding Planner & Bridal Shop

Elegance is a high-end Frontend application built with **React.js**, designed to provide a luxury user experience for wedding coordination and bridal shopping. The project features a sophisticated product engine, real-time media management, and a modern, responsive interface.

---

## 🌐 Live Demo

Experience the application live: **(https://wedding-planner-9aed4.web.app/)**

---

## 🚀 Key Frontend Features

- **Premium Slim Navbar**: A high-performance, sticky navigation system featuring a dynamic cart counter and user profile integration.
- **Advanced Product Engine (Bridal Shop)**:
  - **Live Search**: Instant product discovery by name or unique ID.
  - **Multi-Criteria Filtering**: Filter products by Category, Rating (4.0+ stars), and a dynamic Price Range slider.
  - **Smart Sorting**: Sort items by Price (High to Low / Low to High) and Best Rated.
- **Custom Pagination**: Optimized data fetching and rendering that slices the product grid into 8-10 items per page for faster load times.
- **Cloudinary Integration**: Direct-to-cloud image uploading for user profile pictures with immediate UI previews.
- **Firebase Authentication**: Secure user login and sign-up flows with custom profile data persistence.
- **Persistent Cart Logic**: State-driven shopping cart that synchronizes with Local Storage to prevent data loss on refresh.

---

## 🛠️ Technical Stack

- **Core**: React.js (Create React App)
- **Styling**: Tailwind CSS & DaisyUI
- **Icons**: Lucide React
- **State Management**: React Context API & Hooks (`useState`, `useEffect`)
- **Networking**: Fetch API / Axios
- **Media**: Cloudinary API
- **Auth**: Firebase Authentication
- **Deployment**: Firebase
